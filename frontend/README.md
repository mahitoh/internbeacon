# InternBeacon — frontend

Next.js 16 + React 19 + Tailwind CSS v4. The canonical project documentation lives in the **[repository root `README.md`](../README.md)**.

## Commands

```bash
npm install
npm run dev    # http://localhost:3000
npm run build
npm start
npm run lint
```

## Entry points

- **App**: `app/`
- **Shared components**: `src/components/`
- **Route index (dev)**: [http://localhost:3000/dev/pages](http://localhost:3000/dev/pages)

For structure, layouts, design tokens, and deployment, see the root README.
